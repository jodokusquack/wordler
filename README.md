<h1 align="center">🤖 The Wordler 🤖</h1>

A Wordle Telegram bot for collecting statistics and making leaderboards in group chats.

## 🛠️ Implementation ##
The project is coded in Python and uses the [python-telegram-bot](python-telegram-bot.org) package with the async/await syntax.
The bot works by parsing messages for a Wordle result, autmatically generated by the "Share" feature from Wordle.

It does not store or care about the solution to each puzzle only the amount of guesses needed to solve the puzzle each day.
